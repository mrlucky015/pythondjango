Manage Linux Servers using Webpages

Python-Django Project - WebSite is in Progress

Monitor Servers

Monitor Servers is used to obtain basic information from servers 

Example:
    Date
    Uptime
    VM Statistics
    File Systems
    Memory

Please use below Options for Demo:
    Server name: localhost
    User name: demouser

Users

Users is used to obtain available users and groups on servers 

Please use below Options for Demo:
    Server name: localhost
    User name: demouser
    Add user: provide values in ":" delimited
              username:x:uid:gid:gecos:homedirectory:shell

Packages

Packages is used to list available packages on servers and also list specific package information Packages also installs package on server 

Please use below Options for Demo:
    Server name: localhost
    User name: demouser
    Show Package Info: Provide package name for which information should be displayed
    Install Package: Provide package name which needs to be installed on server

FileSystems

File Systems is used to display file systems available on servers File Systems is also used to create New file systems on servers 

Please use below Options for Demo:
    Server name: localhost
    User name: demouser
    New File System: Provide ":" Delimited values to create file system
    DiskName:ParititionName:PhysicalVolume:VolumeGroup:LogicalVolume:Type:Size:MountPoint

Services

Services is used to Start/Stop/Restart or provide status of server. 

Please use below Options for Demo:
    Server name: localhost
    User name: demouser
    Server Name: provide service name Ex:sshd

Run Command on Server

Run Command is used to run a command on server and provide output 

Please use below Options for Demo:
    Server name: localhost
    User name: demouser
    Enter Command: Provide command which needs to executed on server in "," separated values
